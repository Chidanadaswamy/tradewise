# 🎯 Watchlist - Quick Visual Guide

## Super Simple Steps

### STEP 1️⃣: Open Trade Tab
```
Navigation Menu
├─ Dashboard
├─ Trade ← CLICK HERE
├─ Watchlist
└─ Journal

You'll see:
AAPL - Apple Inc.        $195.50      +2.45%
MSFT - Microsoft Corp    $420.75      -1.20%
GOOGL - Google Inc.      $185.30      +0.50%
```

---

### STEP 2️⃣: Click Bookmark on Stock
```
AAPL - Apple Inc.        $195.50      +2.45%      [☐]
                                                      ↑
                                            CLICK HERE
                                            
After clicking:
AAPL - Apple Inc.        $195.50      +2.45%      [☑]
                                                      ↑
                                         Now FILLED IN
                                         
Success Message: "AAPL added to watchlist!" ✓
```

---

### STEP 3️⃣: View Your Watchlist
```
Navigation Menu
├─ Dashboard
├─ Trade
├─ Watchlist ← CLICK HERE (New!)
└─ Journal

You'll see your saved stocks as cards:

┌─────────────────────────────┐
│ AAPL                    [✕] │  ← Remove button
├─────────────────────────────┤
│ Apple Inc.                  │
├─────────────────────────────┤
│ Current Price: $195.50      │
│ Daily Change: 📈 +2.45%     │
│ Added: Today                │
├─────────────────────────────┤
│ [Trade] [Remove]            │
└─────────────────────────────┘
```

---

### STEP 4️⃣: Monitor Live Prices
```
Your watchlist updates automatically every 5 seconds:

Time: 9:00 AM          Time: 9:05 AM       Time: 9:10 AM
AAPL $190.25          AAPL $190.50        AAPL $191.00
↓                      ↓                    ↓ (prices change)
+0.13%                +0.27%              +0.58%
```

---

### STEP 5️⃣: Remove Stock
```
Option A - From Trade View:
Click the filled bookmark again: [☑] → [☐]
Stock removed from watchlist

Option B - From Watchlist View:
Click [X] button on stock card: [✕]
Click "Yes, remove"
Stock removed from watchlist
```

## 📊 What Each Part Means

### Bookmark Icon
```
☐ = Not in watchlist (click to add)
☑ = In your watchlist (click to remove)
```

### Stock Card in Watchlist
```
┌─────────────────────────────────┐
│ AAPL  (Ticker Symbol)       [X] │ ← Remove button
├─────────────────────────────────┤
│ Apple Inc.                      │ ← Company name
├─────────────────────────────────┤
│ Current Price: $195.50          │ ← Live price (updates every 5s)
│ Daily Change: 📈 +2.45%         │ ← % change today (🟢 green = up, 🔴 red = down)
│ Added: June 18, 2026            │ ← When you added it
├─────────────────────────────────┤
│ [Trade] [Remove]                │ ← Action buttons
└─────────────────────────────────┘
```

## 🎮 Example Usage

### Scenario 1: Add Multiple Stocks
```
1. Trade tab → AAPL bookmark ☐ → ☑ (Added!)
2. Trade tab → MSFT bookmark ☐ → ☑ (Added!)
3. Trade tab → GOOGL bookmark ☐ → ☑ (Added!)
4. Watchlist tab → See 3 cards with all stocks
5. Watch prices update in real-time
```

### Scenario 2: Remove and Re-add
```
1. Watchlist tab → MSFT card
2. Click [X] button → "Remove?" → Yes
3. MSFT disappears from watchlist
4. Change mind? Go to Trade tab
5. MSFT bookmark is empty ☐
6. Click it → ☑ (Added back!)
```

### Scenario 3: Quick Trade
```
1. Watchlist tab → See AAPL at $195.50
2. Interested? Click [Trade] button
3. Goes to Trade view with AAPL selected
4. Buy/Sell form is ready
5. Enter quantity and order
6. Done!
```

## 🆘 Troubleshooting

### Problem: Can't find bookmark button
**Solution:** Look to the right of each stock name in Trade tab

### Problem: Stock not appearing in watchlist
**Solution:** Check if you successfully added it (button should be filled ☑)

### Problem: Prices aren't updating
**Solution:** Check internet connection, page may need refresh

### Problem: Forgot I added a stock
**Solution:** Go to Watchlist tab, you'll see all your stocks

### Problem: Want to remove all stocks
**Solution:** Go to Watchlist, click [X] on each one
